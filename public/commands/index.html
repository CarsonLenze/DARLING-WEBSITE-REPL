Commands

Here you will find a complete list of every command and what they do. You will also be able to find out how to use each command with each section containing the usage and some also containing an example of how someone might use the command!

Note: Any command usage marked with a "?" is an optional argument!

/about

Usage: /about

This command has our vote link, a link to invite the bot, and other information about the bot and it's authors!

/ban

Usage: /ban [user] [?reason]

The ban command is used to Ban a user if they are not following the rules or are misbehaving

​

Example: /ban @C𝕒𝕣𝕤𝕠𝕟 not following the rules

/help

Usage: /help [?catagory]

The help command is used to Show all available commands and how to use them!

Example: /help moderation

/kick

Usage: /kick [user] [?reason]

The kick command is used to kick a user if they are not following the rules or are misbehaving.

Example: /kick @C𝕒𝕣𝕤𝕠𝕟 not following the rules

/mute

Usage: /mute [user] [?reason]

The mute command is used to Mute a user if they are not following the rules or are misbehaving.

Example: /mute @C𝕒𝕣𝕤𝕠𝕟 spamming

/pinfo

Usage: /pinfo [?user]

The profile command is used to view statistics about yourself or another user.

Example: /pinfo @C𝕒𝕣𝕤𝕠𝕟

/ping

Usage: /ping

This command just has information about the bot's uptime and current latency! Useless to some people.




/purge

Usage: /purge [ammount]

The purge command is used to clear out upto 100 messages. This is helpful if someone was spamming.

Example: /purge 100

/tempban

Usage: /tempban [user] [time] [?reason]

The purge command is used to clear out upto 100 messages. This is helpful if someone was spamming.

Example: /tempban @C𝕒𝕣𝕤𝕠𝕟 1d not following the rules

/tempmute

Usage: /tempmute [user] [time] [?reason]

The tempmute command is used to Temporary Mute a user if they are not following the rules or are misbehaving

Example: /tempmute @C𝕒𝕣𝕤𝕠𝕟 1h spamming

/unban

Usage: /unban [user]

The unban command is used to revoke a users Ban.

Example: /unban @C𝕒𝕣𝕤𝕠𝕟

/unmute

Usage: /unmute [user]

The unmute command is used to allow the user to speak again.

Example: /unmute @C𝕒𝕣𝕤𝕠𝕟
